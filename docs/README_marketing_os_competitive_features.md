# Marketing OS Competitive Feature Documents Index

## الغرض
هذه الحزمة تضيف إلى مشروع Marketing OS الخصائص المناسبة المستخلصة من مشاريع:
- erxes
- Medusa
- Strapi
- IDURAR ERP CRM
- Frappe CRM
- NextCRM
- Mattermost

الهدف ليس نسخ هذه المشاريع، بل تحويل الخصائص المناسبة إلى نطاق تنفيذي محكوم داخل Marketing OS.

## القرار التنفيذي
- يبدأ التنفيذ من Phase 0 بتثبيت الحوكمة والحدود ومصدر الحقيقة.
- يدخل Core V1 فقط ما يشغّل دورة التسويق كاملة:
  Understand → Plan → Generate → Review → Publish → Measure → Learn.
- أي خاصية لا تخدم هذه الدورة تؤجل أو ترفض.
